Highlighted Projects

Some projects that I have completed either in my own time or for university work.

Swimclub Project

A web application I created for a unit for my MSc. This application uses a database with users and stores information related to swimming. Users can compare themsleves against eachother and certain users like Managers have elevated priveleges.

MyNote Android Project

This project is a java based android application that uses Google Firestore to store notes under specific users. This application also detects the users battery life and changes the theme to a powersaving one when possible. The application is also secured by the users fingerprint not allowing anyone to access the app unless they are authorised.

Student Portal Project

This project is a java based web application that allows students to view their timetable and set up meetings with their tutors. The teachers can set up meetings, view currently booked ones, edit meetings and delete future meetings.
